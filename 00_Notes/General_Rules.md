## General Rules

- double qutes ""
- := keyword : used when declaring a variable where the type should be inferred by Go
  - example using `:=` keyword:
    ```go
    helloWorld := "Hello World"
    ```
    example using `=` keyword:
    ```go
    var helloWorld string = "Hello World"
    ```
